# Problem Sizes

These are the problem sizes configured in `scripts/ex2_problem_sizes.csv`. For PETSc `ex2`, the mathematical unknown count is `N = m * n` for the 2D five-point finite-difference grid.

| scale | m | n | unknowns | approximate size |
| --- | ---: | ---: | ---: | --- |
| small | 1,000 | 1,000 | 1,000,000 | 1M |
| medium-small | 2,240 | 2,240 | 5,017,600 | 5M |
| medium | 3,160 | 3,160 | 9,985,600 | 10M |
| large | 4,500 | 4,500 | 20,250,000 | 20.25M |
| very-large | 6,400 | 6,400 | 40,960,000 | 41M |
